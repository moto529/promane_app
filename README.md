# README
### プロジェクト名: Promane_app


プログラミング学習を記録していくアプリケーション学習記録をグラフにして、確認することができます。

### 使用技術
* Ruby
  3.1.2
* Ruby on Rails
  7.0.4
* SQLite3
  1.4
* devise
  4.8.1
* Bootstrap
  5.2.2
* Chartkick
  4.2.1
  
### 機能一覧
* ユーザー登録,ログイン機能(devise)

* 投稿機能

  * 言語登録、学習登録機能

* ソート機能(gem無し)

* グラフ機能(Chartkick)
